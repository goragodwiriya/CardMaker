# Card Maker - โปรแกรมออกแบบกราฟิก

Card Maker เป็นเครื่องมือออกแบบกราฟิกเว็บแอปพลิเคชันที่มีลักษณะคล้ายกับ Canva สำหรับสร้างและแก้ไขการออกแบบการ์ด โปสเตอร์ และผลงานกราฟิกอื่น ๆ

## คุณสมบัติหลัก

### 📐 เครื่องมือออกแบบ
- **เทมเพลต** - รับชุดเทมเพลตออกแบบที่พร้อมใช้งาน
- **ข้อความ** - เพิ่มและแก้ไขข้อความด้วยตัวเลือกการจัดรูปแบบที่หลากหลาย
- **รูปทรง** - วาดและจัดการรูปทรงพื้นฐาน (สี่เหลี่ยม วงกลม สามเหลี่ยม เป็นต้น)
- **ไอคอน** - แทรกไอคอนจากไลบรารีไอคอนที่มีอยู่
- **รูปภาพ** - อัปโหลดและจัดการรูปภาพ
- **พื้นหลัง** - ตั้งค่าและแก้ไขพื้นหลังของการออกแบบ

### 🔧 ฟีเจอร์ขั้นสูง
- **QR Code** - สร้างและแทรก QR codes
- **บาร์โค้ด** - สร้างและแทรกบาร์โค้ด
- **ตาราง** - แทรกและแก้ไขตาราง
- **กราฟ** - เพิ่มกราฟและแผนภูมิ
- **เลเยอร์** - จัดการโครงสร้างเลเยอร์ของการออกแบบ

### 💾 การบันทึกและส่งออก
- **บันทึกไฟล์** - บันทึกโครงการของคุณในรูปแบบ JSON
- **ส่งออกหลายรูปแบบ**
  - PNG
  - JPEG
  - SVG
  - PDF
  - JSON (สำหรับการแก้ไขต่อ)
- **พิมพ์** - พิมพ์การออกแบบของคุณโดยตรง

### ⚙️ การทำงาน
- **ย้อนกลับ/ทำซ้ำ** - ย้อนกลับหรือทำซ้ำการเปลี่ยนแปลง (Ctrl+Z / Ctrl+Y)
- **คัดลอก/วาง** - คัดลอกและวางองค์ประกอบ
- **เมนูบริบท** - เมนูคลิกขวาสำหรับการเข้าถึงฟีเจอร์อย่างรวดเร็ว
- **ทีม** - สลับระหว่างธีมแสงและมืด
- **เอกสารหลายหน้า** - จัดการหลายหน้าในโครงการเดียว

## สถาปัตยกรรม

Card Maker ถูกสร้างขึ้นด้วยสถาปัตยกรรม Manager:

```
Application (Root)
├── EventBus - การสื่อสารระหว่างผู้จัดการ
├── Store - ที่เก็บสถานะแบบรวม
├── HistoryManager - บันทึกการเปลี่ยนแปลง
├── DocumentManager - การจัดการเอกสาร
├── PageManager - การจัดการหน้า
├── ObjectManager - จัดการวัตถุการออกแบบ
├── LayerManager - จัดการเลเยอร์
├── SelectionManager - การจัดการการเลือก
├── CanvasManager - การเรนเดอร์พื้นที่ทำงาน
├── ExportManager - ส่งออกไปยังรูปแบบต่าง ๆ
├── ImportManager - นำเข้าจากรูปแบบต่าง ๆ
├── FontManager - จัดการฟอนต์
├── AssetManager - จัดการสินทรัพย์
├── TemplateManager - จัดการเทมเพลต
└── [และ Manager อื่น ๆ...]
```

## การติดตั้ง

Card Maker เป็นแอปพลิเคชันเว็บที่ไม่มีการขึ้นต่อกัน ไม่จำเป็นต้องติดตั้ง

### ข้อกำหนดที่ต้องมี
- เบราว์เซอร์สมัยใหม่ที่รองรับ:
  - HTML5 Canvas
  - ES6+ JavaScript
  - Fetch API

### การใช้งาน
1. เปิด `index.html` ในเบราว์เซอร์ของคุณ
2. หรือวางไว้บนเซิร์ฟเวอร์เว็บและเข้าถึงผ่าน URL

## วิธีการใช้งาน

### สร้างเอกสารใหม่
1. คลิก **"ไฟล์ใหม่"** ในแถบเครื่องมือด้านบน
2. เลือกหรือสร้างเอกสารเปล่า

### เพิ่มเนื้อหา
1. ใช้แถบด้านซ้ายเพื่อเลือกประเภทเนื้อหา (ข้อความ รูปทรง รูปภาพ เป็นต้น)
2. คลิกที่องค์ประกอบเพื่อเพิ่มลงในพื้นที่ทำงาน
3. ลากและปล่อยเพื่อเปลี่ยนตำแหน่ง
4. ใช้แถบด้านขวา (คุณสมบัติ) เพื่อปรับแต่ง

### บันทึกและส่งออก
- **บันทึก**: คลิก **"บันทึก"** เพื่อบันทึกในรูปแบบ JSON
- **ส่งออก**: คลิก **"ส่งออก ▾"** และเลือกรูปแบบ

### คีย์ลัด
| คีย์ลัด | การกระทำ |
|--------|--------|
| Ctrl+Z | ย้อนกลับ |
| Ctrl+Y | ทำซ้ำ |
| Ctrl+C | คัดลอก |
| Ctrl+V | วาง |
| Ctrl+X | ตัด |
| Delete | ลบ |

## เทคโนโลยีที่ใช้

- **HTML5** - โครงสร้าง
- **CSS3** - สไตล์และเลเยาต์
- **Vanilla JavaScript (ES6+)** - ตรรมชาติ
- **Canvas API** - การเรนเดอร์กราฟิก
- **SVG** - ส่งออกเวกเตอร์

## การสนับสนุนเบราว์เซอร์

- Chrome / Chromium 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## หมายเหตุเกี่ยวกับการพัฒนา

### การเพิ่ม Manager ใหม่
1. สร้างไฟล์ใหม่ใน `js/managers/`
2. ทำให้ขยายได้จาก `BaseManager`
3. ลงทะเบียนใน `Application.js`

### การเพิ่มปลั๊กอิน
1. สร้างไฟล์ใหม่ใน `js/plugins/`
2. ทำให้ปฏิบัติตามอินเทอร์เฟส `Plugin`
3. ลงทะเบียนใน `PluginManager`
